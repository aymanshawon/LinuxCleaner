## Usage
##### Usage: Cleaner [-a all] [-c cache ] [-l logs]

#####  -a all:  Clean All Old Data Or Logs
#####  -c cache:  Clean All Apt Cache
#####  -l log:  Clean all Logs
#####  -t temp: Clean Old Datas And Browser Cache Datas
#####  -h help: Help Menu
